# Execute in dev mode

1. Clone the repo
2. Execute

```
npm install
```

3. Nest CLI must be installed

```
npm i -g @nestjs/cli
```

4. Set up the database

```
docker-compose up -d
```

## Used Stack

- MongoDB
- NestJS
